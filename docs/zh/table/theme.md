# 主题样式

-   可根据颜色配置定制自己的主题

## light
  
::: demo 默认是light主题 

theme/light
h:500px
:::


## dark

::: demo 通过config传入自定义颜色配置实现暗黑主题(比css变量更优先)

theme/dark
h:500px
:::
